1. /shop

des: 商店集合, 用于新增商店和取得商店列表(首页分页用)

get:

  • params: ?offset=limit=

  • response: {"shops": [{"business_hours": null, "category_list": "/sdfsdf/asdfasdf/1.txt,sdfsdfsdfsdf", "telephone_no": null, "latitude": null, "city_id": null, "district_id": null, "shop_name": "\u59b9\u5b50\u4f60\u8fd9\u662f\u574f\u6389\u4e86\u4e48", "longititude": null, "shop_id": 1, "pic_url_list": "/sdfsdf/asdfasdf/1.txt,sdfsdfsdfsdf", "address": null, "qrcode": null, "business_area": null}, {"business_hours": null, "category_list": null, "telephone_no": null, "latitude": null, "city_id": null, "district_id": null, "shop_name": "\u5305\u5934\u5b50", "longititude": null, "shop_id": 2, "pic_url_list": null, "address": null, "qrcode": null, "business_area": null}], "total_num": 3}

  • offset: 起始商店位置, limit: 打算获得多少个商店. 返回值说明: total_num 代表系统中总的商店数量而非此次取得的商店数量, 用作分页用.

  • body:
      shop_id, 必填
  shop_name, 必填
  brand_name
  business_hours
  telephone_no
  city_id
  district_id
  business_area
  address
  longititude
  latitude
  category_list

3. /shop/[$shop_id]/introduction

des: 商店图文介绍对象, 用于获取和修改商店图文信息

get:

  • params: $shop_id
  • response: {"introduction": "24333", "shop_id": 1}

put:

  • params: $shop_id
  • body: introduction = 'balabalabalbalalabalbalbalablablabalbal'
  • response: {"is_success": true, "shop_id": 1}

post:

  • 同 put, 用的是一套代码, post代表新增, put代表修改, 提供两个http动词来符合 restful 设计

4. /shop/[$shop_id]/image

des: 表示指定商店对象所包含的图片对象的集合

get

  • params: $shop_id
  • response: {"imgs": ["/fd/images/1.jpg", "/fd/images/2.jpg",], "total_num": 2}
  • 返回该商品所有的图片 url
  • 如果传shop_id = 0, 那么可以打开一个图片上传测试网页, 仅供开发测试使用, 上线将注释掉该功能

post

  • response: {"result": [{"is_success": true, "img_url": "/fdimage/goodsimage/10.jpg", "filename": "test.png"}], "total_num": 1}
  • 该接口仅上传图片, 并未将图片url写入商店图片url列表, 上传完图片后请使用 put 动作更新商店的图片列表
  • 请使用上述 get 方法, 传 goods_id=0 进行测试

put

  • params: $shop_id, ?pic_url_list=
  • response: {"is_success": true, "shop_id": 1}
  • pic_url_list 为必填参数, 传入逗号分隔的 uri 列表.比如 "/fd/image/1.jpg, /fd/image/2.jpg"
  • 更新商品的图片uri列表字段
