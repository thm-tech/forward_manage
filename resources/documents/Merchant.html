Merchant

1. /merchant/[$shop_id]/info

des: 商店账号信息类

get:

params: $shop_id

response: {"contact_phone_no": null, "contact_name": "newbrash\u5927\u738b", "shop_id": 1, "contact_email": null, "contact_qq": null}

put:

  • params: $shop_id
  • body:
      contact_name 
  contact_phone_no
  contact_email
  contact_qq
  • response: {"is_success": true, "shop_id": 1}
  • 修改商店账号的基本信息, body 参数均可为空

2. /merchant/[$shop_id]/serviceinfo

des: 商店账号服务信息类

get:

params: $shop_id

response: {"service_deadline": null, "shop_id": "1", "last_pay_time": null, "service_balance": null, "last_pay_money": null, "service_status": null}

3. /merchant/isexist

des: 判断商店账号是否存在

get:

params: ?account

response: {"is_exist": false}

4. /merchant

des: 商店账号集合, 用于商店账号注册

post

  • body:
      account, 必填
  password, 必填, 仅接受16位或32位md5加密后字符串
  contact_name
  contact_phone_no
  contact_email
  contact_qq
  shop_name, 必填
  brand_name
  business_hours
  telephone_no
  city_id
  district_id
  business_area
  address
  longititude
  latitude
  category_list
  • response: {"is_success": true, "shop_id": 1}
